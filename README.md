# Quarkus gRPC with Buf

A Quarkus gRPC application that replaces the built-in `protoc` code generation with the [Buf](https://buf.build) toolchain, adding package management, linting, breaking change detection, and [Protovalidate](https://github.com/bufbuild/protovalidate) support.

Read more in [Adding Buf to Quarkus'gRPC](https://simplewins.substack.com/p/adding-buf-to-quarkusgrpc) at my [Substack](https://simplewins.substack.com/).
